Comandos no gitbash para atualizar :

git status -> verificar se há alteração nos arquivos
git add -A -> adicionar todos os arquivos alterados e mudanças
git commit -m "(melhoria)" -> comando para informar a melhoria feita ou versão nova.
git push -> faz o upload no github.
